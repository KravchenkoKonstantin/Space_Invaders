Наименование проекта: Space Invaders
Студент Кравченко Константин Алексеевич
Группа: АСУбз-25-1
Цель проета: Создать 2D-аркаду «Space Invaders» с последовательно усложняющимися волнами врагов, боссами и таблицей рекордов.
Краткое описание: Аркадная игра, где игрок управляет космическим кораблём и уничтожает наступающие волны врагов. С каждой волной враги становятся быстрее и сильнее, появляются новые типы, а каждые пять волн — босс. Игра ведёт счёт, сохраняет рекорды и имеет меню с кнопками «Старт», «Топ лидеров» и «Выход». За основу я взял классическую игру Space Invaders 1978 года.
Используемые технологии:
Язык программирования: Python 3.10+
Игровой движок: Pygame / Pygame-ce
Графика: используются готовые спрайты
Хранение данных: текстовый файл в формате JSON для сохранения таблицы рекордов
Управление в игре:
Стрелки влево/вправо — движение корабля в соответствующую сторону.
Пробел — стрельба.
Enter — начать игру заново.
Q — вернуться в главное меню.
